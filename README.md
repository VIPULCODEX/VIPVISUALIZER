# Kidney Exchange Visualization

A web application designed to visualize the kidney exchange dataset, complete with performance benchmarks of different matching algorithms.

## Features
- **Modern UI**: Clean, responsive interface with a dark theme and smooth animations.
- **Graph Visualization**: Visualizes the kidney exchange donor-recipient compatibility graph.
- **Algorithm Performance**: Benchmarks and compares the performance of induced and acyclic matching algorithms.

## Prerequisites
- Python 3.x

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/VIPULCODEX/VIPVISUALIZER.git
   cd VIPVISUALIZER
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application locally:
   ```bash
   python app.py
   ```

## Deployment
The application is configured to be easily deployable to Vercel via the included `vercel.json` configuration.
